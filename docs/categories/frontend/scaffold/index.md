# 一、vue 的使用

## 1、vue-cli

`vue` 项目是基于 `vue-cli` 脚手架搭建的项目。当我们要创建一个项目时，首先要先全局安装 `vue-cli` 脚手架，命令行为：

```bash
npm i -g @vue/cli
```
