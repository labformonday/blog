# JavaScript 全局对象 & 核心类与对象



## 全局对象

当 `JavaScript` 解释器启动时（或者任何Web浏览器加载新页面的时候），它将创建一个新的全局对象，并给他它一组定义的初始属性

- 全局属性，比如 `undefined`、`Infinity和NaN`
- 全局函数，比如`isNaN()`、`parseInt()`和`eval()`
- 构造函数，比如`Date()`、`RegExp()`、`String()`、`Object()`和`Array()`
- 全局对象，比如 `Math` 和 `JSON`
